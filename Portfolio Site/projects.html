Machine Perception: Investigating Visual Data Analysis through CNN Hand Gesture Recognition
This project aimed to enhance human-computer interaction byreplacing traditional methods of visual data analysis (keyboard and mouse) with psychologically intuitive hand-gesture controls, investigating the concept of visualizing multivariate datasets as expansible stars in a 3D space.

This project was done as part of my 3rd-year undergraduate module 'Undergraduate Individual Project'.

I received an overall grade of 89% in this project.
